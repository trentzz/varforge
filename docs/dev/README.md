# Developer Guide

## Build

```bash
cargo build --release
```

The release binary is at `target/release/varforge`.
No system dependencies are required. All I/O libraries are pure Rust.

## Test

```bash
cargo test --all-targets
cargo clippy --all-targets -- -D warnings
cargo fmt -- --check
```

## Run

```bash
# Validate a config
varforge validate --config examples/minimal.yaml

# Run a simulation
varforge simulate --config examples/wgs_30x.yaml

# Spike variants into an existing BAM
varforge edit --bam input.bam --vcf variants.vcf --output output.bam

# Learn quality profile from a real BAM
varforge learn-profile --bam real.bam --output profile.json
```

## Benchmarks

Benchmarks require reference FASTA files in `benchmarking/`.
Synthetic references are generated by `benchmarking/scripts/gen_refs.py`.
For hg38 benchmarks, download chr22:

```bash
cd benchmarking
wget https://hgdownload.soe.ucsc.edu/goldenPath/hg38/chromosomes/chr22.fa.gz
gunzip chr22.fa.gz
```

Run benchmarks:
```bash
bash benchmarking/scripts/run_benchmarks.sh
```

Generate graphs:
```bash
python3 benchmarking/scripts/plot_results.py
```
