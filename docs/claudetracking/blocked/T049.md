# T049: Bundle hg38 microsatellite loci BED as an optional static asset

**Epic**: EPIC-ADVANCED (overallplans/EPIC-ADVANCED.md)
**Priority**: low
**Depends on**: T048
**Status**: blocked

## Goal

Bundle a compressed hg38 microsatellite loci BED file as an optional static asset. When `tumour.msi: true` and no `msi_bed` path is specified, the bundled BED is used automatically.

## Steps

1. Obtain or generate an hg38 microsatellite loci BED file.
2. Compress and embed using `include_bytes!` or a build script.
3. In the MSI path from T048, when `msi_bed` is not specified, decompress and use the bundled BED.
4. Allow users to override with a custom BED via `tumour.msi_bed: <path>`.

## Notes

Used when `msi_bed` not specified; compressed and embedded in binary.
