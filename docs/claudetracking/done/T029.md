# T029: Embed SBS96 signature matrix as compressed static asset

**Epic**: EPIC-SIGNATURES (overallplans/EPIC-SIGNATURES.md)
**Priority**: high
**Depends on**: none
**Status**: done

## Goal

The COSMIC v3.4 SBS96 signature matrix is embedded in the binary as a compressed static asset. It is decompressed and parsed at runtime when signature-weighted mutation generation is requested.

## Steps

1. Download the COSMIC v3.4 SBS96 matrix (96 trinucleotide channels × N signatures).
2. Compress and embed using `include_bytes!` or a build script.
3. Write a parser that produces a `HashMap<SigName, [f64; 96]>` at runtime.
4. Expose a function to retrieve a normalised probability vector for a given signature name.

## Notes

Source: COSMIC v3.4. 96 trinucleotide channels × N signatures.
