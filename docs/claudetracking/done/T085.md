# T085: Validate UMI length < read_length

**Epic**: EPIC-CONFIG-VALIDATION
**Priority**: P1
**Depends on**: none
**Status**: done

## Goal

Add a validate() check that `umi.length < read_length`. An inline UMI longer than or equal
to the read length would consume the entire read, leaving no sequence.

## Steps

1. In validate(), if `cfg.umi.is_some()`:
   `ensure!(umi.length < cfg.read_length, "umi.length ({}) must be < read_length ({})", ...)`
2. Add a unit test: umi.length=150, read_length=150 fails validation
3. `cargo test`
