# T116: Normalise platform name case for GATK compatibility

**Priority**: P1
**Depends on**: —
**Status**: todo

## Goal

The `PL` tag in the BAM `@RG` header must be uppercase for GATK compatibility
(e.g. `ILLUMINA`, not `illumina` or `Illumina`). Normalise the platform name
to uppercase when writing the BAM header.

## Steps

1. Locate where the `PL` field is written to the BAM header.
2. Apply `.to_uppercase()` (or equivalent) to the platform value.
3. Add a test that checks the `PL` value in a generated BAM header is uppercase.
4. `cargo fmt -- --check && cargo clippy -- -D warnings && cargo test`

## Success criteria

- [ ] `PL` tag value is uppercase in all BAM output.
- [ ] A test asserts the uppercase format.
- [ ] `cargo test` passes cleanly.
