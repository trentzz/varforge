# T039: Implement chromothripsis model (random cuts + re-ligation on one chromosome)

**Epic**: EPIC-ADVANCED (overallplans/EPIC-ADVANCED.md)
**Priority**: medium
**Depends on**: T037, T038
**Status**: done

## Goal

A chromothripsis model simulates random cuts and re-ligation events on a single chromosome, producing a characteristic pattern of complex rearrangements. Triggered by `sv_signature: chromothripsis`.

## Steps

1. Implement a `ChromothripsisSvGenerator` that selects one chromosome and applies N random cuts.
2. Randomly re-ligate the resulting fragments (in random order and orientation).
3. Generate the resulting SV calls (deletions, inversions, translocations).
4. Accept `sv_signature: chromothripsis` with a configurable fragment count.

## Notes

`sv_signature: chromothripsis`; configurable fragment count.
