# T020: Use stored per-target multipliers in `coverage_multiplier_at`

**Epic**: EPIC-COVERAGE (overallplans/EPIC-COVERAGE.md)
**Priority**: medium
**Depends on**: T019
**Status**: todo

## Goal

`coverage_multiplier_at` uses the stored per-target depth when available, and falls back to log-normal sampling when no depth is stored for a given target.

## Steps

1. In `coverage_multiplier_at`, look up the target key in the depth map.
2. If found, return the stored multiplier directly.
3. If not found, use the existing log-normal sampling path.

## Notes

Fall back to log-normal sampling when no stored depth for a target.
