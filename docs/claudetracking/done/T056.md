# T056: Extract seq_utils module with complement and reverse_complement

**Epic**: EPIC-REFACTOR
**Priority**: P2
**Depends on**: none
**Status**: done

## Goal

Create `src/seq_utils.rs` (or `src/core/seq_utils.rs`) with a single canonical
`complement(base: u8) -> u8` and `reverse_complement(seq: &[u8]) -> Vec<u8>`. Remove the
duplicate copies in `src/variants/signatures.rs`, `src/variants/structural.rs`, and
`src/core/engine.rs`.

## Steps

1. Create the module with the two functions and unit tests
2. Add `pub mod seq_utils;` to the appropriate mod.rs
3. Replace all duplicate definitions with `use crate::seq_utils::{complement, reverse_complement}`
4. Confirm `cargo test` still passes

## Notes

Three separate private implementations exist. They should be identical, but any future divergence
would produce subtly different output depending on which code path was taken.
