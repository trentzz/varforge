# T027: Single-read BAM output mode (no R2, template length = read length)

**Epic**: EPIC-LONGREAD (overallplans/EPIC-LONGREAD.md)
**Priority**: high
**Depends on**: none
**Status**: done

## Goal

Long-read platforms produce single-end reads. This task adds a BAM output mode that omits R2, sets template length equal to read length, and clears paired-end flags.

## Steps

1. Add a `single_read` output mode flag, set automatically for long-read platforms.
2. In the BAM writer, when `single_read` is set, omit R2 records.
3. Set `TLEN` to read length and clear `0x1` (paired), `0x2` (proper pair), and mate flags.

## Notes

Detect via platform. Omit R2 fields from BAM records.
