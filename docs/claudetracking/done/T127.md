# T127: Integration tests for cancer presets

**Priority**: P1
**Depends on**: —
**Status**: done

## Goal

There are no integration tests that exercise cancer presets (e.g. `twist`,
`ctdna`, others). Add tests that run VarForge with each built-in preset and
assert that simulation completes without error and produces plausible output.

## Steps

1. Identify all built-in cancer presets.
2. For each preset, write an integration test that:
   - Runs the simulation with the preset config.
   - Asserts output files are created.
   - Asserts `sim_report.json` contains expected fields.
3. `cargo fmt -- --check && cargo clippy -- -D warnings && cargo test`

## Success criteria

- [ ] Each built-in preset has at least one integration test.
- [ ] Tests confirm output files and `sim_report.json` are produced.
- [ ] `cargo test` passes cleanly.
