# T064: Boundary tests for ffpe.rs end_enrichment

**Epic**: EPIC-TESTS
**Priority**: P2
**Depends on**: none
**Status**: done

## Goal

Add unit tests in `src/artifacts/ffpe.rs` for the `end_enrichment` function covering boundary
positions: pos=0, pos=length-1, length=1, and a position in the interior.

## Steps

1. `length=1, pos=0`: the only position — verify no panic and return is in [0.0, 1.0]
2. `length=100, pos=0`: start of read — verify enrichment is at maximum (or near-maximum)
3. `length=100, pos=99`: end of read — same as pos=0 by symmetry
4. `length=100, pos=50`: interior — enrichment should be at minimum
5. Confirm no integer underflow (all arithmetic stays in bounds)
6. `cargo test`
