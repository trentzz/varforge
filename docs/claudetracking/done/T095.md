# T095: Add --list-presets flag to simulate subcommand

**Epic**: EPIC-CLI-ERGONOMICS
**Priority**: P2
**Depends on**: none
**Status**: done

## Goal

Add a `--list-presets` flag to the simulate subcommand that prints all available base presets
and cancer presets then exits. Users should not need to read source code to discover preset
names.

## Steps

1. Add `--list-presets` boolean flag to `SimulateOpts` in `src/cli/mod.rs`
2. In `src/cli/simulate.rs`, check `opts.list_presets` before doing anything else and print:
   - Base presets (from `src/cli/presets.rs`): small, panel, wgs, cfdna, ffpe, umi
   - Cancer presets (from `src/cli/cancer_presets.rs`): luad, brca, crc, etc.
   - For each: name and one-line description
3. Print to stdout and return Ok(())
4. Add a test that calls run() with list_presets=true and checks the output contains "panel"
5. `cargo test`
