# T079: Support homozygous genotypes (1/1) in truth VCF

**Epic**: EPIC-TRUTH-VCF
**Priority**: P1
**Depends on**: none
**Status**: done

## Goal

`src/io/truth_vcf.rs:117` hardcodes `GT=0/1` for all variants. Germline homozygous variants
(VAF ≥ 0.99) should emit `GT=1/1`. Somatic clonal variants at VAF ≈ purity/2 should remain
`0/1`.

## Steps

1. In the truth VCF writer, check `variant.expected_vaf >= 0.99` (or use clone_id containing
   "hom") to select `1/1` vs `0/1`
2. For germline variants with clone_id "germline_hom", always emit `1/1`
3. Add a unit test: germline_hom variant produces GT=1/1; somatic variant produces GT=0/1
4. `cargo test`
