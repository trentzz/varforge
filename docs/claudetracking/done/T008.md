# T008: Write batch manifest listing all VAF runs

**Epic**: EPIC-BATCH (overallplans/EPIC-BATCH.md)
**Priority**: high
**Depends on**: T007
**Status**: done

## Goal

After all VAF runs complete, write a TSV manifest to the top-level output directory. The manifest records the parameters and output location for each run.

## Steps

1. Accumulate a manifest row after each VAF run completes.
2. Write `batch_manifest.tsv` to `{output}/` with columns: `vaf`, `output_dir`, `n_variants`, `seed`.
3. Ensure the manifest is written even if individual runs fail (record failure status).

## Notes

TSV columns: `vaf`, `output_dir`, `n_variants`, `seed`.
