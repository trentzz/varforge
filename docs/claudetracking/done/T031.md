# T031: Signature-weighted alt base selection in random mutation generator

**Epic**: EPIC-SIGNATURES (overallplans/EPIC-SIGNATURES.md)
**Priority**: high
**Depends on**: T029, T030
**Status**: done

## Goal

Replace uniform alt base selection in the random mutation generator with a weighted draw from the SBS96 probability vector for the variant's trinucleotide context and the configured signature.

## Steps

1. In the random mutation generator, look up the trinucleotide context via T030.
2. Retrieve the signature probability vector via T029 for the configured signature name.
3. Draw the alt base using the weighted probabilities for the three possible substitutions in that context.
4. Ensure the drawn base differs from the reference base.

## Notes

Config field: `mutations.random.signature: "SBS1"` (or similar). Replace uniform base choice with weighted draw.
