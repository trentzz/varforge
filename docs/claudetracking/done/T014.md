# T014: Append variant tag string to FASTQ read name on write

**Epic**: EPIC-OUTPUT (overallplans/EPIC-OUTPUT.md)
**Priority**: high
**Depends on**: T013
**Status**: done

## Goal

When writing FASTQ output, append the variant tag string to the read name for each read pair that carries a variant. This makes the mutation visible directly in the read name.

## Steps

1. In the FASTQ writer, check `read_pair.variant_tags`.
2. For each tag, format the string ` VT:Z:{chrom}:{pos}:{vartype}` and append to the read name.
3. If multiple tags are present, append all of them.

## Notes

Format: ` VT:Z:chr7:55191822:SNV` appended to the read name.
