# T010: Compute and print expected alt molecule count table

**Epic**: EPIC-BATCH (overallplans/EPIC-BATCH.md)
**Priority**: high
**Depends on**: T009
**Status**: done

## Goal

In dry-run mode, compute and print a table of expected alt molecule counts per VAF. Exit with code 1 if any VAF yields fewer than 1 expected alt molecule family.

## Steps

1. For each VAF (or each entry in `vafs`), compute: `expected_families = coverage × vaf / family_size_mean`.
2. Print a table to stdout: `vaf`, `expected_alt_molecules`, `expected_alt_families`.
3. Exit 1 if any row has `expected_alt_families < 1`.

## Notes

Formula: `coverage × vaf / family_size_mean`. Exit 1 if any VAF yields < 1 expected family.
