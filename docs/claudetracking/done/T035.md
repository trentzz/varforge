# T035: Implement `PairedSimulationMode`: shared germline, split somatic spike-in

**Epic**: EPIC-GERMLINE (overallplans/EPIC-GERMLINE.md)
**Priority**: high
**Depends on**: T032, T034
**Status**: done

## Goal

`PairedSimulationMode` runs two simulations that share a germline variant set. The normal sample receives germline variants only. The tumour sample receives germline plus somatic variants.

## Steps

1. Generate the shared germline variant set once.
2. Run the normal simulation: spike in germline variants only; write to `normal/` subdir.
3. Run the tumour simulation: spike in germline + somatic variants; write to `tumour/` subdir.
4. Write a combined manifest listing both output directories.

## Notes

Normal receives germline only; tumour receives germline + somatic.
