# T024: Add `LongReadFragmentSampler` (log-normal, 5–50 kbp)

**Epic**: EPIC-LONGREAD (overallplans/EPIC-LONGREAD.md)
**Priority**: high
**Depends on**: none
**Status**: done

## Goal

A new `LongReadFragmentSampler` samples fragment lengths from a log-normal distribution with a mean and standard deviation configurable in kbp, covering the 5–50 kbp range typical of long-read sequencing.

## Steps

1. Define `LongReadFragmentSampler` as a new variant of `FragmentSampler`.
2. Accept `mean_kbp` and `sd_kbp` parameters in the config.
3. Sample from a log-normal distribution in the `sample` method.
4. Clamp output to a configurable `[min_len, max_len]` range.

## Notes

Configurable mean/sd in kbp.
