**Depends on**: T006, T008

## Goal

A `benchmark-suite` subcommand generates a full NĂ—M grid of configs spanning VAF and coverage parameters, runs all combinations, and writes a manifest linking output directories to parameter sets.

## Steps

1. Define the `benchmark-suite` subcommand with `--vafs`, `--coverages`, and `--output` arguments.
2. Generate one config per (VAF, coverage) combination using the batch mode from T006/T008.
3. Run all configs (sequentially or in parallel).
4. Write a manifest TSV: `vaf`, `coverage`, `output_dir`, `seed`.
