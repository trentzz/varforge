# T136: Validate stochastic VAF with chi-squared test

**Epic**: EPIC-RELEASE-GATE (overallplans/EPIC-RELEASE-GATE.md)
**Priority**: high
**Depends on**: none
**Status**: todo

## Goal

The paper claims stochastic Bernoulli VAF sampling produces observed VAF that
follows binomial variance. The evaluation section acknowledges this is "not yet
independently validated." Add a quantitative test that generates reads at known
VAFs and verifies the observed alt-read counts match a binomial distribution
using a chi-squared goodness-of-fit test.

## Success Criteria

- [ ] An integration test or script generates reads at multiple VAFs (0.05, 0.1, 0.25, 0.5) at 100x depth.
- [ ] For each VAF, the observed alt-read count distribution across >= 100 variants is tested against Binomial(depth, VAF) using chi-squared.
- [ ] All chi-squared p-values > 0.01 (no significant deviation from expected distribution).
- [ ] Results are either in a test file or a script with saved output in `benchmarking/results/`.
- [ ] `/update` has been run after changes.

## Steps

1. Write an integration test that creates a minimal config with known variants at specific VAFs.
2. Run the simulation engine and collect per-variant alt counts.
3. Compute the chi-squared statistic against the expected binomial distribution.
4. Assert p > 0.01.
