# T100: Replace hardcoded reference paths in examples with ${reference}

**Epic**: EPIC-EXAMPLES
**Priority**: P2
**Depends on**: none
**Status**: done

## Goal

Every example YAML config in `examples/` hardcodes `/data/ref/hg38.fa` (or similar). Replace
all occurrences with `${reference}` so users can run examples with
`--set reference=/path/to/hg38.fa` without editing the files.

## Steps

1. Find all reference path strings across `examples/*.yaml`
2. Replace each with `${reference}`
3. Similarly replace any hardcoded output directories with `${output_dir}` if present
4. Update any comments that describe the path to describe the variable instead
5. Confirm the example configs still parse correctly with `cargo test` (config load tests)
