# T028: Wire platform detection into engine and fragment/quality selection

**Epic**: EPIC-LONGREAD (overallplans/EPIC-LONGREAD.md)
**Priority**: high
**Depends on**: T016, T024, T025, T026, T027
**Status**: done

## Goal

Setting `platform: "pacbio-hifi"` or `platform: "nanopore-r10"` in the config automatically selects the correct fragment sampler, quality model, and BAM output mode.

## Steps

1. Read `platform` from config in the engine initialisation path.
2. Map platform names to the correct `FragmentSampler`, `QualityModel`, and output mode.
3. Assert that unknown platform names produce a clear error at startup.
4. Add integration tests for each supported platform.

## Notes

`platform: "pacbio-hifi"` selects correct sampler + quality model. Preset T016 must be complete.
