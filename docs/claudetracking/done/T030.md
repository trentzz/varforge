# T030: Implement trinucleotide context lookup from reference sequence

**Epic**: EPIC-SIGNATURES (overallplans/EPIC-SIGNATURES.md)
**Priority**: high
**Depends on**: none
**Status**: done

## Goal

Given a variant position, extract the trinucleotide context (bases at positions -1, 0, +1) from the pre-fetched region sequence. This context indexes into the SBS96 probability vector.

## Steps

1. Write a `trinucleotide_context(region_seq: &[u8], pos: usize) -> [u8; 3]` function.
2. Handle boundary cases (pos == 0 or pos == region_len - 1).
3. Normalise to the pyrimidine convention (C or T at the central base) as per COSMIC SBS96.
4. Return the 96-channel index for lookup.

## Notes

Extract [−1, 0, +1] context from pre-fetched `region_seq`. Must follow COSMIC SBS96 pyrimidine convention.
