# T094: Migrate from serde_yaml 0.9 to serde_yml

**Epic**: EPIC-DEPENDENCY-HYGIENE
**Priority**: P2
**Depends on**: none
**Status**: done

## Goal

`serde_yaml 0.9` is unmaintained (the author archived the repo). The community-maintained
fork is `serde_yml`. This is a near-drop-in replacement.

## Steps

1. In Cargo.toml, replace `serde_yaml = "0.9"` with `serde_yml = "0.0"` (latest)
2. Find all `use serde_yaml::` imports and replace with `use serde_yml::`
3. Check for any API differences: `serde_yml::from_str`, `serde_yml::to_string`, error types
4. `cargo build && cargo test`

## Notes

The main API difference is the crate name; the function signatures are compatible. Error types
may differ slightly — adjust match arms if needed.
