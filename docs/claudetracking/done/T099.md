# T099: Share germline variant set across multi-sample runs

**Epic**: EPIC-MULTI-SAMPLE
**Priority**: P1
**Depends on**: T098
**Status**: done

## Goal

In multi-sample (longitudinal) mode, each sample currently generates its own independent
germline variant set. For a single patient across timepoints, the germline must be identical.
Generate germline variants once and share across all samples.

## Steps

1. In `run_multi_sample`, generate the germline variant list once before entering the
   per-sample loop
2. Pass the shared germline list to each sample's variant builder so it is not re-generated
3. Each sample's somatic variants are still generated independently
4. Add a test: 3-sample multi-sample run, confirm all three truth VCFs contain identical
   germline variants (same positions)
5. `cargo test`
