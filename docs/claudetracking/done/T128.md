# T128: Integration tests for duplex UMI mode

**Priority**: P1
**Depends on**: —
**Status**: done

## Goal

There are no integration tests covering duplex UMI simulation end-to-end. Add
tests that run VarForge in duplex UMI mode and assert correct tag output,
strand family assignment, and duplex conversion rate.

## Steps

1. Write an integration test config with duplex UMI mode enabled.
2. Run the simulation and assert:
   - `RX:Z` tag is present on all read records.
   - Strand families (AB/BA) are correctly assigned.
   - Duplex conversion rate in `sim_report.json` is within expected range.
3. `cargo fmt -- --check && cargo clippy -- -D warnings && cargo test`

## Success criteria

- [ ] Integration test covers duplex UMI mode.
- [ ] `RX:Z` tag is present in output reads.
- [ ] Duplex conversion rate is reported in `sim_report.json`.
- [ ] `cargo test` passes cleanly.
