# T110: Verify ultra-low VAF Poisson sampling is unbiased at VAF < 0.01%

**Epic**: EPIC-TWIST-DUPLEX
**Priority**: P1
**Depends on**: none
**Status**: todo

## Goal

At 2000× coverage and VAF = 0.0001 (0.01%), the expected number of alt-carrying molecules
is 0.2 per region — meaning most runs produce 0 alt molecules and occasional runs produce 1.
Verify the Bernoulli sampling in `sample_alt_count()` is unbiased at these extreme values
and does not floor at 0 or exhibit off-by-one errors.

## Steps

1. Write a statistical test: run `sample_alt_count(depth=2000, vaf=0.0001)` 100,000 times.
   The mean should converge to 0.2 ± 0.05. A systematic bias (mean < 0.15 or > 0.25) would
   indicate a rounding or floor error.
2. Also test `depth=5000, vaf=0.00001` (expected 0.05 alt molecules): mean should converge
   to 0.05 ± 0.02
3. Check for off-by-one: at `vaf=1.0, depth=100`, alt_count should always be 100
4. If biases are found, fix the sampling in `src/core/engine.rs` (`sample_alt_count`)
5. Add the statistical tests to the test suite with a seed for reproducibility
6. `cargo test`

## Notes

At ultra-low VAFs, even a small multiplicative bias (e.g. always 1.5× the true VAF) would
make benchmarks systematically misleading. The test passes if the 95% confidence interval
on the mean includes the expected value.
