# T037: Implement HRD deletion pattern generator (100 kbp–10 Mbp deletions with LOH)

**Epic**: EPIC-ADVANCED (overallplans/EPIC-ADVANCED.md)
**Priority**: medium
**Depends on**: none
**Status**: todo

## Goal

A homologous recombination deficiency (HRD) SV signature generator produces large deletions (100 kbp to 10 Mbp) with loss of heterozygosity, triggered by `sv_signature: HRD` in the mutations block.

## Steps

1. Implement an `HrdSvGenerator` that samples deletion positions and sizes from the HRD-typical distribution.
2. Add LOH annotation to generated deletions.
3. Integrate with the existing SV spike-in path.
4. Accept `sv_signature: HRD` in the mutations config block.
