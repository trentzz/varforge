# T117: Fix NM:i: tag to reflect actual mismatches for variant reads

**Priority**: P1
**Depends on**: —
**Status**: done

## Goal

The `NM:i:` tag should count the actual number of mismatches and indel bases
for each read. For variant reads (SNV, indel), the current value may not
reflect the introduced variant. Fix the tag to be computed from the actual
sequence and CIGAR.

## Steps

1. Identify where `NM:i:` is set in read construction.
2. Compute NM from the read sequence, reference, and CIGAR after applying variants.
3. Add a test that generates variant reads and asserts the NM value is correct.
4. `cargo fmt -- --check && cargo clippy -- -D warnings && cargo test`

## Success criteria

- [ ] `NM:i:` reflects the true mismatch count for both reference and variant reads.
- [ ] A test covers SNV and indel cases.
- [ ] `cargo test` passes cleanly.
