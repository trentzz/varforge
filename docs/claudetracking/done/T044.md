# T044: Add `capture.mode: amplicon`; implement fixed-length fragment sampling at amplicon coords

**Epic**: EPIC-TOOLING (overallplans/EPIC-TOOLING.md)
**Priority**: low
**Depends on**: none
**Status**: done

## Goal

In amplicon mode, fragments exactly span each amplicon defined in the capture BED. No off-target reads are generated. Fragment length is fixed by the amplicon coordinates.

## Steps

1. Add `capture.mode: amplicon` to the config schema.
2. In amplicon mode, for each amplicon in the BED, generate fragments that exactly span the amplicon.
3. Suppress off-target fragment sampling entirely.
4. Ensure coverage is distributed uniformly across amplicons (modulo the per-target depth from T019 if set).

## Notes

Fragments exactly span each amplicon; no off-target reads.
