# T040: Add `contamination` config block; implement synthetic contamination from SNP profile VCF

**Epic**: EPIC-ADVANCED (overallplans/EPIC-ADVANCED.md)
**Priority**: medium
**Depends on**: none
**Status**: done

## Goal

A `contamination` config block enables synthetic cross-sample contamination. Reads carrying donor SNPs from a profile VCF are drawn at the specified fraction.

## Steps

1. Define a `contamination` config section with fields: `fraction`, `source: snp_profile`, `vcf` path.
2. Load donor SNPs from the VCF.
3. During fragment generation, at the configured fraction, sample a fragment and spike in donor SNPs.
4. Ensure contamination reads are not included in the truth VCF.

## Notes

Draw reads carrying donor SNPs at specified `fraction`.
