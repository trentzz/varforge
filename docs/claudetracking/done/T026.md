# T026: Add Nanopore R10 quality model (Q15–Q25, homopolymer errors)

**Epic**: EPIC-LONGREAD (overallplans/EPIC-LONGREAD.md)
**Priority**: high
**Depends on**: none
**Status**: done

## Goal

A Nanopore R10 quality model produces qualities in the Q15–Q25 range with context-dependent homopolymer error injection.

## Steps

1. Define a `NanoporeR10QualityModel` implementing the quality model trait.
2. Sample base qualities in the Q15–Q25 range.
3. Detect homopolymer runs in the reference sequence and inject insertion/deletion errors at elevated rate within runs.
4. Register the model under the `nanopore-r10` preset.

## Notes

Position-independent quality, but context-dependent homopolymer error injection.
