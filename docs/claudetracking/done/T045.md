# T045: Implement `primer_trim` option for amplicon reads

**Epic**: EPIC-TOOLING (overallplans/EPIC-TOOLING.md)
**Priority**: low
**Depends on**: T044
**Status**: done

## Goal

When `primer_trim` is set, trim N bases matching the primer sequence from each read end and mark the trimming in the BAM record.

## Steps

1. Add `capture.primer_trim: <int>` config field.
2. In the amplicon read writer, trim the specified number of bases from each read end.
3. Set the soft-clip CIGAR operation for trimmed bases in the BAM record.
4. Verify trimmed reads align correctly to the amplicon interior.

## Notes

Trim N bases matching primer from read ends; mark in BAM.
