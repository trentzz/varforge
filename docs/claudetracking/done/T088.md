# T088: Validate mutation VCF path exists at config load time

**Epic**: EPIC-CONFIG-VALIDATION
**Priority**: P1
**Depends on**: none
**Status**: done

## Goal

If `mutations.vcf_path` is set, check the file exists in validate(). A typo in the path
currently fails at simulation time with a generic I/O error after potentially minutes of
reference loading.

## Steps

1. In validate(), if `mutations.vcf_path.is_some()`, check `path.exists()`
2. Also accept `.vcf.gz` and confirm the `.tbi` index exists alongside it
3. Return a clear error naming the missing file
4. Add a unit test: config with a nonexistent vcf_path fails validation
5. `cargo test`
