# T021: Track per-chromosome coverage statistics (mean, sd) during simulation

**Epic**: EPIC-COVERAGE (overallplans/EPIC-COVERAGE.md)
**Priority**: medium
**Depends on**: none
**Status**: todo

## Goal

The simulation accumulates per-chromosome coverage statistics (mean and standard deviation) as it processes regions. These are finalised after all regions complete and made available to the QC report writer.

## Steps

1. Add a per-chromosome coverage accumulator to the simulation state.
2. Increment the accumulator for each fragment placed in each region.
3. Compute mean and sd per chromosome after all regions complete.

## Notes

Accumulate in region loop; finalise after all regions complete.
