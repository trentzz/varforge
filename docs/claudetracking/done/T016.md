# T016: Define preset lookup table in io/config.rs

**Epic**: EPIC-PRESETS (overallplans/EPIC-PRESETS.md)
**Priority**: high
**Depends on**: none
**Status**: done

## Goal

A static lookup table in `io/config.rs` maps preset names (e.g. `illumina-umi`, `pacbio-hifi`) to their default parameter structs.

## Steps

1. Define a `Preset` enum or string-keyed map in `io/config.rs`.
2. Populate with at least the core chemistry presets and their default values.
3. Expose a `lookup_preset(name: &str) -> Option<PresetDefaults>` function.
