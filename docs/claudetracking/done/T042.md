# T042: Add 4-mer end motif frequency table; implement rejection sampling in fragment generation

**Epic**: EPIC-ADVANCED (overallplans/EPIC-ADVANCED.md)
**Priority**: medium
**Depends on**: none
**Status**: todo

## Goal

cfDNA fragments have non-uniform 4-mer end motif frequencies. This task adds an empirical frequency table and uses rejection sampling to accept only fragments whose end motifs match the configured profile.

## Steps

1. Embed the empirical 4-mer end motif frequency table from plasma cfDNA literature.
2. Add `fragment.end_motif_model: "plasma"` config option.
3. In fragment generation, after selecting a start position, check the 4-mer at each end against the frequency table.
4. Use rejection sampling: accept the fragment with probability proportional to its end motif frequency; otherwise resample.

## Notes

`fragment.end_motif_model: "plasma"`. Empirical frequencies from literature.
