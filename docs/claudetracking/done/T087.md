# T087: Warn when purity=1.0 with cfDNA fragment model

**Epic**: EPIC-CONFIG-VALIDATION
**Priority**: P2
**Depends on**: T076
**Status**: done

## Goal

After T076 fixes the ctDNA fraction formula, emit a tracing::warn when
`fragment.model == cfDNA && purity == 1.0`. This combination means 100% of fragments are
tumour-derived with a flat fragment length distribution, which is biologically unrealistic.

## Steps

1. In the engine, after resolving ctdna_fraction, add:
   `if ctdna_fraction == 1.0 && is_cfdna { tracing::warn!("purity=1.0 with cfDNA model: all fragments will be tumour-derived. Did you mean to set a lower purity?") }`
2. Confirm the warning appears in tests when VARFORGE_LOG=warn
