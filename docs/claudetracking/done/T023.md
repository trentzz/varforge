# T023: Write `sim_report.json` with coverage, VAF accuracy, family stats

**Epic**: EPIC-COVERAGE (overallplans/EPIC-COVERAGE.md)
**Priority**: medium
**Depends on**: T021, T022
**Status**: todo

## Goal

After simulation completes, write `sim_report.json` to the output directory. The report summarises per-chromosome coverage, per-variant actual VAF, and family size statistics.

## Steps

1. Collect coverage stats from T021 and family stats from T022.
2. Compute per-variant actual VAF from observed alt molecule counts.
3. Serialise to `sim_report.json` using serde_json.
4. Write to `{output}/sim_report.json`.

## Notes

JSON structure: per-chrom coverage (mean, sd), per-variant actual VAF, family size stats (mean, sd, histogram).
