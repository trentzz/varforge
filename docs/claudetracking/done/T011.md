Accumulate counts of alt molecules and duplex alt molecules per variant as the simulation processes all regions, so the totals are available for writing to the truth VCF.

## Steps

1. Add per-variant counters for `n_alt_mol` and `n_duplex_alt` in the simulation state.
2. Increment counters during spike-in for each read pair carrying the variant.
3. Finalise counts after all regions complete.
