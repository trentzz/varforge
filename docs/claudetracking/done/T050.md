## Goal

The `learn-profile` command reads a BAM file and builds a 2D quality-by-cycle count matrix (`[cycle][quality_bin]`) from the query quality strings of aligned reads.

## Steps

1. Implement or complete the `learn-profile` subcommand.
2. Stream reads from the input BAM via noodles.
3. For each read, iterate over query qualities and increment the `[cycle][quality_bin]` counter.
4. Normalise each cycle's counts to a probability distribution.
