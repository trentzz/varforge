# T032: Add `germline` config section; implement population-frequency SNP/indel generator

**Epic**: EPIC-GERMLINE (overallplans/EPIC-GERMLINE.md)
**Priority**: high
**Depends on**: none
**Status**: done

## Goal

A `germline` section in the YAML config enables SNP and indel generation at population-frequency densities. Germline variants are distributed uniformly across the genome unless a VCF is supplied.

## Steps

1. Define the `germline` config section with fields `het_snp_density` and `hom_snp_density` (per kbp), and optional `vcf` path.
2. Implement a generator that samples germline SNP/indel positions at the specified densities.
3. Assign heterozygous variants VAF = 0.5 and homozygous variants VAF = 1.0.
4. If a VCF is supplied, use its variants instead of sampling.

## Notes

`het_snp_density` and `hom_snp_density` per kbp. Uniform across genome unless VCF supplied.
