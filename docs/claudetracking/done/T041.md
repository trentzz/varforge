# T041: Implement BAM-source contamination (randomly subsample reads from donor BAM)

**Epic**: EPIC-ADVANCED (overallplans/EPIC-ADVANCED.md)
**Priority**: medium
**Depends on**: T040
**Status**: todo

## Goal

In addition to SNP-profile contamination, support drawing contamination reads directly from a donor BAM file by random subsampling.

## Steps

1. Extend the `contamination` config section with `source: bam` and a `bam` path field.
2. Open the donor BAM via noodles and stream reads.
3. At the configured fraction, randomly subsample donor reads and inject them into the output.
4. Ensure injected reads are excluded from truth VCF.

## Notes

Requires streaming reads from external BAM via noodles.
