# T033: Write `germline_truth.vcf` output

**Epic**: EPIC-GERMLINE (overallplans/EPIC-GERMLINE.md)
**Priority**: high
**Depends on**: T032
**Status**: done

## Goal

A separate `germline_truth.vcf` file records all germline variants used in the simulation, with VAF = 0.5 for heterozygous and VAF = 1.0 for homozygous variants.

## Steps

1. Collect all germline variants from the generator in T032.
2. Write `germline_truth.vcf` to the output directory using the noodles VCF writer.
3. Set the AF INFO field to 0.5 or 1.0 as appropriate.
4. Include standard VCF header lines.

## Notes

Separate file from the somatic truth VCF. VAF = 0.5 (het) or 1.0 (hom).
