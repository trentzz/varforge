# T139: Update outdated dependencies

**Epic**: EPIC-POLISH (overallplans/EPIC-POLISH.md)
**Priority**: medium
**Depends on**: T133
**Status**: todo

## Goal

Several dependencies have newer major versions available. Update them to
current versions. Key updates: `criterion` 0.5 to 0.8, `indicatif` 0.17 to
0.18. Also evaluate noodles crate updates if available.

## Success Criteria

- [ ] `cargo outdated` shows no dependencies with newer compatible versions (Compat column).
- [ ] `cargo test` passes after updates.
- [ ] `cargo bench` compiles (benchmarks do not need to be re-run).
- [ ] `cargo clippy -- -D warnings` passes.
- [ ] `/update` has been run after changes.

## Steps

1. Run `cargo outdated` and review each entry.
2. Update `Cargo.toml` versions for crates with newer compatible releases.
3. For major version bumps (criterion, indicatif), check changelogs for breaking changes.
4. Fix any compilation errors from API changes.
5. Run full test suite.

## Notes

The `number_prefix` unmaintained advisory will resolve automatically when
`indicatif` is updated to 0.18 (which drops `number_prefix`).
