# T097: Suggest closest preset name when unknown preset is specified

**Epic**: EPIC-CLI-ERGONOMICS
**Priority**: P3
**Depends on**: none
**Status**: todo

## Goal

When the user specifies `--preset pannel` (typo), the error should say "unknown preset
'pannel'; did you mean 'panel'?" rather than a terse None/unwrap failure.

## Steps

1. Add a simple edit-distance function (Levenshtein, ≤ 30 lines) or use the `strsim` crate
2. In the preset lookup, if name is not found, find the closest match among all preset names
3. If edit distance ≤ 2, include "did you mean '<name>'?" in the error
4. Add a test: preset name "pannel" suggests "panel"
5. `cargo test`
