# T093: Audit and remove unused dependencies

**Epic**: EPIC-DEPENDENCY-HYGIENE
**Priority**: P2
**Depends on**: none
**Status**: todo

## Goal

Identify and remove crates in Cargo.toml that are not actually used in non-test code. Reduce
compile time and supply-chain surface.

## Steps

1. Run `cargo +nightly udeps` or manually audit each dep by grepping for its use
2. For each potentially unused crate, confirm by removing it and running `cargo build`
3. Pay special attention to: `bio`, `statrs`, `thiserror`, individual noodles sub-crates
4. Remove unused crates from Cargo.toml
5. For crates used only in tests, move them to `[dev-dependencies]`
6. `cargo build && cargo test`
