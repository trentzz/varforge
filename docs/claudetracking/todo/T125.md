# T125: Integration tests for copy number alterations

**Priority**: P1
**Depends on**: —
**Status**: todo

## Goal

There are no integration tests covering copy number alteration (CNA) simulation.
Add tests that run VarForge with a CNA config and assert that read depth
reflects the specified copy number in affected regions.

## Steps

1. Write an integration test config with one or more CNA regions.
2. Run the simulation and assert:
   - Read depth in CNA regions reflects the specified copy number ratio.
   - Truth output records the expected CNA regions.
3. `cargo fmt -- --check && cargo clippy -- -D warnings && cargo test`

## Success criteria

- [ ] Integration test covers at least one CNA region.
- [ ] Observed depth ratio in the CNA region is consistent with config.
- [ ] `cargo test` passes cleanly.
