# T057: Unify variant position and type helpers as methods on MutationType

**Epic**: EPIC-REFACTOR
**Priority**: P2
**Depends on**: none
**Status**: todo

## Goal

Replace the three free functions `variant_position` (engine.rs), `mutation_pos` (engine.rs),
and `variant_pos` (truth_vcf.rs) with a single `MutationType::position(&self) -> u64` method.
Do the same for the vartype free functions.

## Steps

1. Add `position(&self) -> u64` to `MutationType` in `src/core/types.rs`
2. Add `vartype_str(&self) -> &'static str` to `MutationType` (consolidating the vartype helpers)
3. Update all call sites in engine.rs, truth_vcf.rs, simulate.rs
4. Delete the free functions
5. `cargo test`

## Notes

Having three free functions with slightly different names for the same operation is a maintenance
hazard — callers may use the wrong one or add a fourth.
