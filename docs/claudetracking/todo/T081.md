# T081: Write bgzip-compressed truth VCF with tabix index

**Epic**: EPIC-TRUTH-VCF
**Priority**: P1
**Depends on**: none
**Status**: todo

## Goal

Compress the truth VCF with bgzip and write a `.tbi` tabix index alongside it so that
bcftools, IGV, and other standard tools can use it without manual post-processing.

## Steps

1. Use `noodles-bgzf` (already in Cargo.toml) to wrap the truth VCF writer in a BGZF encoder
2. Change the output file extension from `.vcf` to `.vcf.gz`
3. After writing, build a tabix index using `noodles-tabix` and write `<output>.vcf.gz.tbi`
4. Update `OutputConfig` path resolution to append `.vcf.gz`
5. Update any test that checks for a plain `.vcf` output path
6. `cargo test`
