# T123: Integration tests for paired tumour-normal mode

**Priority**: P1
**Depends on**: —
**Status**: todo

## Goal

There are no integration tests covering paired tumour-normal simulation. Add
tests that run VarForge with a tumour-normal config and assert correct output
for both samples.

## Steps

1. Write an integration test config specifying a tumour and matched normal.
2. Run the simulation and assert:
   - Separate output files are created for tumour and normal.
   - Variants appear in tumour but not normal reads.
   - Truth VCF contains the expected somatic variants.
3. `cargo fmt -- --check && cargo clippy -- -D warnings && cargo test`

## Success criteria

- [ ] Integration test covers tumour and normal outputs.
- [ ] Somatic variants are absent from normal output.
- [ ] `cargo test` passes cleanly.
