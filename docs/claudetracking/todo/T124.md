# T124: Integration tests for capture/panel mode

**Priority**: P1
**Depends on**: —
**Status**: todo

## Goal

There are no integration tests covering capture or panel simulation mode. Add
tests that run VarForge with a capture BED config and assert that output reads
are restricted to target regions.

## Steps

1. Write an integration test config with a capture BED file (can be synthetic).
2. Run the simulation and assert:
   - Output reads fall within target regions.
   - On-target rate matches config expectations.
   - `sim_report.json` includes capture metrics.
3. `cargo fmt -- --check && cargo clippy -- -D warnings && cargo test`

## Success criteria

- [ ] Integration test uses a capture BED and checks on-target read placement.
- [ ] `sim_report.json` includes capture uniformity metrics.
- [ ] `cargo test` passes cleanly.
