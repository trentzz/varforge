# T061: Integration tests for cli/edit.rs end-to-end

**Epic**: EPIC-TESTS
**Priority**: P1
**Depends on**: none
**Status**: todo

## Goal

Add integration tests in `tests/` (or as `#[test]` in `src/cli/edit.rs`) that exercise the
full edit pipeline: load a tiny BAM, load a VCF, spike variants, write output BAM, verify
the output contains the expected mutations.

## Steps

1. Create a minimal test BAM (20–30 paired reads over a 500 bp region) using noodles in
   the test helper — no external files needed
2. Create a minimal VCF with one SNV in that region
3. Run `cli::edit::run()` with an EditOpts pointing at the test BAM and VCF
4. Read the output BAM and verify at least one read has the alt base at the variant position
5. Test purity: at purity=0.5 with 20 reads over the position, roughly half carry the alt

## Notes

The edit subcommand has zero integration tests. Any regression in the full pipeline would go
undetected.
