\section{Conclusion}
\label{sec:conclusion}

\varforge{} is a unified framework for synthetic cancer sequencing data generation.
It combines de-novo reads, variant injection, tumour modelling, UMI/duplex simulation, cfDNA fragmentation, and library artefacts in a single Rust binary.
The streaming architecture bounds memory independently of dataset size, making full human genome simulation feasible on standard workstations.
Source code, documentation, and example configurations are available under the MIT licence at \url{https://github.com/trentzz/varforge}.
